Welcome to the server panel!

Get Started

You are at your server ip with the port: 4000

If you are an admin RMTpass in the config.yml contains the credentials to access console remotely

If your developer wants to implement features into a bot/website make sure you give them APIpass from the config.yml

To prevent abuse you can update the BlockedCmds in the config.yml with a list of commands you do not want to run

Custom page/html

To have custom html here, please contact a web developer or use a premade html/css/js scripts and put it in the html folder

If there are any issues feel free to reach out to the plugin developer

Api development

This plugin comes with an api that can be accessed by passing a valid APIpass

To prevent abuse you can update the BlockedCmds in the config.yml with a list of commands you do not want to run
